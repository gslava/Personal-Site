# Project Overview

This repository contains a personal website built with:

- `Angular 21`
- `TypeScript`
- static-first deployment targets:
  - `Cloudflare Pages`
  - `GitHub Pages`

The site currently has:

- a shared top navigation/header
- localized routes
- a Home page with a reference-driven particle effect around the centered text `BUILD`
- placeholder section pages for:
  - `Technologies`
  - `Projects`
  - `About Me`
  - `Contact`

# Working Rules

1. The current actionable implementation plan always lives in [PLANS.MD](/Users/viacheslav.guzhov/Documents/IdeaProjects/Personal%20Site/PLANS.MD).

2. When a new plan is created, the previous contents of `PLANS.MD` must be fully replaced.

3. The particle effect must be implemented from documented mechanics, not from screenshot imitation alone.

4. Keep the implementation clean-room:

- do not copy the reference bundle verbatim
- do reproduce the same class of behavior and interaction

5. Preserve the existing Angular app structure unless the current plan explicitly requires broader changes.
